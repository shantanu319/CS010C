Lewis Carrol (The "Alice in Wonderland" guy) developed these...

Start with simple pairs like

wheat
bread

It will give you a list like:
  wheat cheat cleat bleat bleak break bread
where each word is one letter different than the preceding word

Make sure you can do the edge cases like

wheat
wheat

Others to test are

reads
brats

yards   <this one doesn't exist!)
augur
